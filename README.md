El archivo [detector_flancos_bajada.vhd](https://github.com/raulvarandela/HAE-T7/blob/master/detector_flancos_bajada.vhd) corresponde a un circuito detector de flancos de bajada. Dicho circuito se caracteriza porque su salida (s) se pone a 1 durante un tiempo dado (Tclk) cada vez que detecte que la señal de entrada (e) ha descrito un flanco de bajada. A continuación se muestra un ejemplo de la relación entre la señal de entrada y la señal de salida del circuito a diseñar.
